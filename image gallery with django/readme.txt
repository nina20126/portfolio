Teachers requirements (in finnish):

Web-palvelu, jossa on sisäänkirjautuminen ja käyttäjätunnusten luonti (hyödynnä Djangon valmiita autentikointipalveluita).

Käyttäjä voi ladata bittikarttakuvia palvelimelle eri kansioihin, ainakin seuraavissa tiedostomuodoissa: JPEG, PNG ja GIF. Järjestelmä luo automaattisesti JPEG-esikatselukuvakkeen (thumbnail) kuvasta ja osaa skaalata kuvat sopivaan resoluutioon. Tämä kannattaa toteuttaa hyödyntäen esim. Pillow-kirjastoa ja samalla kun käyttäjä lataa palvelimelle uusia kuvia, järjestelmä tallentaa samana kuvan usealla eri resoluutiolla palvelimelle. Käyttäjä voi myös tarkastella ja poistaa kuvia.
Onnistuuko kuvien järjestäminen?

English:

So the assingment was to create a web service using django where user can upload bitmap images to different folders. It should accept JPEG, PNG ang GIF formats. Program should create thumbnail images automaticly when user uploads image. User can create an account, log in and log out, examine images and their infos and delete images. Optional challenge was give user a possibility to organize images and folders. 

This was a group assingment. Also Mira Hämäläinen and Hanna Sinkkilä was creating this program.

We managed to reach all teachers requirements and also optional task to organize images.
